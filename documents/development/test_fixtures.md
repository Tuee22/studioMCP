> **Purpose**: Define the deterministic fixture set used by adapter validation, fixture seeding, and example DAG coverage in `studioMCP`.

## Summary

Deterministic fixtures are generated under `.data/studiomcp/generated-fixtures/` and can be
mirrored into MinIO bucket `studiomcp-test-fixtures`.

## CLI Workflow

```bash
docker compose run --rm studiomcp studiomcp test seed-fixtures
docker compose run --rm studiomcp studiomcp test verify-fixtures
```

- `test seed-fixtures` generates missing local fixtures, ensures the bucket exists, and uploads all registered artifacts
- `test verify-fixtures` compares deterministic local bytes against MinIO objects by SHA-256

## Manifest

`examples/fixtures/manifest.yaml` is the repo-owned index for the current fixture inventory and is
kept aligned with `src/StudioMCP/Test/Fixtures.hs`.
